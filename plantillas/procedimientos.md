# Plantilla para escribir manuales de procedimientos

Formato recomendado para escribir manuales de procedimientos en Markdown.

<br/>

```markdown
# Guía acerca de ""tema"" para ""perfil del lector""

_**Fecha:** 00/00/0000 **Versión del sistema:** 0.0.0 **Autor:** nombre <contacto@example.com>_

## Introducción
<!-- Explica el marco conceptual del procedimiento -->

## Requisitos
<!-- Lista los requisitos para realizar el procedimiento -->

## Resultado
<!-- Describe el resultado esperado -->

## Advertencias
<!-- Lista las advertencias que el lector debe tener en cuenta -->

## Procedimiento
<!--
Explica el procedimiento paso a paso en un flujo principal
Escribe flujos de excepción en caso de que el procedimiento contemple múltiples caminos para llegar al resultado
Escribe flujos alternos en caso de que el procedimiento contemple múltiples resultados o casos de error
-->

## Referencias
<!-- Lista documentos que refuercen o complementen la información del procedimiento -->
[markdown reference](https://example.com)
```
